# Ejecutar en desarrollo ‚ñ∂Ô∏è

1. Clonar repo.
2. Ejecutar .

```
cd pokedex

npm i

```

3. Tener Nest CLI instalado.

```
npm i -g @nestjs/cli
```

4. Levantar la base de datos.

```
docker-compose up -d
```

5. Correr servidor local.

```
npm run start:dev
```

6. Recargar base de datos con seed.

```
GET http://localhost:3000/api/v2/seed
```
